# Fashion Dataset Practical — Theory & Viva Questions

Typical Fashion-MNIST practicals: image classification with CNNs, data augmentation, model evaluation.

Viva Q&A:
- Q: What preprocessing is needed for image datasets?
  - A: Rescaling (0–1), resizing to a common input shape, and optionally normalization using dataset mean/std.
- Q: How to interpret model accuracy on validation vs test?
  - A: Validation accuracy guides hyperparameter tuning; final test accuracy reports generalization.
- Q: Why use `ImageDataGenerator`/augmentation?
  - A: Augmentation increases training data variety and reduces overfitting.
